# framerix-privacy
프레임릭스(FrameRix) 개인정보처리방침

프레임릭스는 사용자의 신체 측정 기능 제공을 위해 카메라를 사용합니다.

1. 수집하는 정보
본 애플리케이션은 어떠한 개인정보도 수집하거나 저장하지 않습니다.

2. 데이터 처리
모든 데이터는 사용자의 기기 내에서만 처리되며 외부 서버로 전송되지 않습니다.

3. 권한 사용
카메라 권한은 신체 측정 기능을 위해서만 사용됩니다.

4. 제3자 제공
어떠한 사용자 데이터도 제3자에게 제공되지 않습니다.
